# VPS-рЃќрЃћ Discord Bot-рЃўрЃА рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ

## ­ЪЊІ рЃгрЃўрЃюрЃљрЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃў

VPS-рЃќрЃћ рЃБрЃюрЃЊрЃљ рЃњрЃЦрЃЮрЃюрЃЊрЃћрЃЌ:
- Ubuntu/Debian Linux рЃАрЃћрЃарЃЋрЃћрЃарЃў
- SSH рЃгрЃЋрЃЊрЃЮрЃЏрЃљ
- Root рЃљрЃю sudo рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃў

## ­Ъџђ рЃюрЃљрЃЉрЃўрЃ»-рЃюрЃљрЃЉрЃўрЃ» рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ

### 1. VPS-рЃќрЃћ рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ

```bash
ssh root@your_vps_ip
# рЃљрЃю
ssh your_username@your_vps_ip
```

### 2. рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

```bash
sudo apt update && sudo apt upgrade -y
```

### 3. Bun Runtime-рЃўрЃА рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃћрЃЉрЃљ

```bash
curl -fsSL https://bun.sh/install | bash
```

рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ Bun PATH-рЃерЃў:

```bash
echo 'export BUN_INSTALL="$HOME/.bun"' >> ~/.bashrc
echo 'export PATH="$BUN_INSTALL/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

рЃњрЃљрЃЊрЃљрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ:

```bash
bun --version
```

### 4. Git-рЃўрЃА рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃћрЃЉрЃљ (рЃЌрЃБ рЃљрЃа рЃљрЃарЃўрЃА)

```bash

```

### 5. рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ VPS-рЃќрЃћ

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў A: Git Repository-рЃЊрЃљрЃю (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў)**

рЃЌрЃБ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў GitHub-рЃќрЃћрЃљ:

```bash
cd ~
git clone https://github.com/your-username/geotiers-bot.git
cd geotiers-bot
```

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў B: рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃ«рЃћрЃџрЃўрЃЌ рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ**

рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃџрЃЮрЃЎрЃљрЃџрЃБрЃа рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃќрЃћ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ:

```bash
# Windows PowerShell-рЃЊрЃљрЃю
scp -r C:\Users\shaka\OneDrive\Desktop\geotiers-bot root@your_vps_ip:/root/
```

рЃљрЃю рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ FileZilla/WinSCP рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА.

### 6. Environment Variables-рЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

VPS-рЃќрЃћ рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `.env` рЃцрЃљрЃўрЃџрЃў:

```bash
cd ~/geotiers-bot
nano .env
```

рЃЕрЃљрЃАрЃЋрЃўрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў (рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃарЃћрЃљрЃџрЃБрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃЌ):

```env
DISCORD_TOKEN=your_actual_discord_token
CLIENT_ID=your_actual_client_id
GUILD_ID=your_actual_guild_id
```

**рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў:** рЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў `.env` рЃцрЃљрЃўрЃџрЃўрЃЊрЃљрЃю token-рЃў, рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃА рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃЏрЃЪрЃдрЃљрЃЋрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў token Discord Developer Portal-рЃќрЃћ:
1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ https://discord.com/developers/applications
2. рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ
3. Bot Рєњ Reset Token Рєњ рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЌ рЃљрЃ«рЃљрЃџрЃў token

рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ: `Ctrl + X`, рЃерЃћрЃЏрЃЊрЃћрЃњ `Y`, рЃерЃћрЃЏрЃЊрЃћрЃњ `Enter`

### 7. Dependencies-рЃўрЃА рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃћрЃЉрЃљ


### 9. рЃЉрЃЮрЃбрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

рЃ»рЃћрЃа рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃЉрЃЮрЃбрЃў рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ рЃарЃћрЃЪрЃўрЃЏрЃерЃў:

рЃЌрЃБ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃЎрЃљрЃарЃњрЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА, рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ `Ctrl + C` рЃерЃћрЃАрЃљрЃЕрЃћрЃарЃћрЃЉрЃџрЃљрЃЊ.

### 10. PM2-рЃўрЃА рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃћрЃЉрЃљ (Process Manager)

PM2 рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЉрЃЮрЃбрЃўрЃА рЃЏрЃБрЃЊрЃЏрЃўрЃЋ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА:

```bash
# Node.js-рЃўрЃА рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃћрЃЉрЃљ (PM2-рЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃљ)
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

# PM2-рЃўрЃА рЃЊрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃћрЃЉрЃљ

```

### 11. рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ PM2-рЃўрЃЌ

```bash

```

### 12. PM2 Auto-Start рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃарЃЮрЃЏ рЃЉрЃЮрЃбрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃћрЃерЃЋрЃљрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃарЃћрЃАрЃбрЃљрЃарЃбрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ:

```bash

```

## ­ЪћД рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ PM2 рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў

```bash
# рЃЉрЃЮрЃбрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ

# рЃџрЃЮрЃњрЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ

# рЃЉрЃЮрЃбрЃўрЃА рЃарЃћрЃАрЃбрЃљрЃарЃбрЃў

# рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃЕрЃћрЃарЃћрЃЉрЃљ
pm2 stop geotiers-bot

# рЃЉрЃЮрЃбрЃўрЃА рЃгрЃљрЃерЃџрЃљ PM2-рЃЊрЃљрЃю
pm2 delete geotiers-bot

# рЃДрЃЋрЃћрЃџрЃљ рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ

# рЃџрЃЮрЃњрЃћрЃЉрЃўрЃА рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ
pm2 flush
```

## ­Ъћё рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

рЃарЃЮрЃфрЃљ рЃЎрЃЮрЃЊрЃА рЃерЃћрЃфрЃЋрЃџрЃўрЃЌ:

```bash

# Git-рЃўрЃЊрЃљрЃю рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ (рЃЌрЃБ рЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ Git-рЃА)

# рЃљрЃю рЃ«рЃћрЃџрЃўрЃЌ рЃљрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃў

# Dependencies-рЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ (рЃЌрЃБ package.json рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ)
bun install

# Commands-рЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ (рЃЌрЃБ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃў рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ)

# рЃЉрЃЮрЃбрЃўрЃА рЃарЃћрЃАрЃбрЃљрЃарЃбрЃў
```


2. **рЃџрЃЮрЃњрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ:**
PM2 рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃџрЃЮрЃњрЃћрЃЉрЃА `~/.pm2/logs/` рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў

3. **рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ:**
рЃЌрЃБ рЃњрЃАрЃБрЃарЃЌ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃЉрЃЮрЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ, рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃфрЃљрЃџрЃЎрЃћ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ:
```bash
pm2 start "bun run start" --name bot1
pm2 start "bun run start" --name bot2
```
